# Employee Management System

A complete Employee Management System built using **Python**, **Streamlit**, **SQLite**, **Pandas**, and **Plotly**.

## Features

### Employee Management
- Add Employee
- Edit Employee
- Delete Employee
- Search Employees

### Attendance Management
- Mark Attendance
- Attendance Report
- Attendance Percentage
- Low Attendance Alerts

### Salary Management
- Generate Salary
- Salary History
- Salary Slip
- Salary Raise Tracking

### HR Analytics Dashboard
- KPI Cards
- Department-wise Employee Count
- Salary Distribution
- Gender Distribution
- Department Salary Analysis
- Department Payroll Analysis
- Top 5 Highest Paid Employees

## Tech Stack

- Python
- Streamlit
- SQLite
- Pandas
- Plotly
- Git & GitHub
